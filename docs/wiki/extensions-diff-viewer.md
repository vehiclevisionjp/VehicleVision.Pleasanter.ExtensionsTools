## 概要

`ExtensionsDiffViewer` は、CLI ツール（`ExtensionsSyncTool`）と同じ Common ライブラリを使用して、サーバーとローカルの差分比較をデスクトップ GUI で行えるアプリケーションです。

主な特徴:

- **左右ペイン表示**: サーバー側とローカル側のコンテンツを並べて比較
- **差分ステータスの色分け**: 一致・変更あり・サーバーのみ・ローカルのみを視覚的に識別
- **差分を見ながら同期**: 個別の拡張機能を選択して Pull/Push 操作が可能
- **全件一括操作**: すべての拡張機能をまとめて Pull/Push

---

## 前提条件

- .NET SDK 10.0 以上
- プリザンター 1.3.13.0 以上（Extensions API が有効であること）
- プリザンターの API キー

---

## 起動方法

```bash
dotnet run --project src/ExtensionsDiffViewer
```

## 画面構成

### 設定パネル（上部）

| 項目        | 説明                                       |
| ----------- | ------------------------------------------ |
| Base URL    | プリザンターサーバーのベース URL           |
| API Key     | プリザンター API キー                      |
| Parameters  | ローカルの Parameters ディレクトリのパス   |
| 比較ボタン  | サーバーとローカルの差分比較を実行         |
| 全件 Pull ↓ | サーバーの全レコードをローカルに書き出す   |
| 全件 Push ↑ | ローカルの全ファイルをサーバーに登録・更新 |

### 拡張機能一覧（左ペイン）

差分比較の結果が一覧表示されます。各アイテムには:

- **色付きインジケーター**: 差分ステータスに応じた色
- **種別名**: `[ExtensionType] ExtensionName` 形式
- **ステータスラベル**: 「一致」「変更あり」「サーバーのみ」「ローカルのみ」

### 差分ビューア（右ペイン）

選択した拡張機能のコンテンツが左右に並んで表示されます:

- **左側（青）**: サーバー（DB）のコンテンツ
- **右側（紫）**: ローカル（ファイル）のコンテンツ

上部に個別の Pull/Push ボタンがあります。

### ステータスバー（下部）

- 現在の操作状態メッセージ
- 各ステータスの件数サマリー

## 操作方法

### 差分比較

1. 上部の設定パネルに Base URL、API Key、Parameters パスを入力
2. 「比較」ボタンをクリック
3. 左ペインに差分一覧が表示される
4. 一覧からアイテムを選択すると、右ペインにサーバー・ローカルのコンテンツが並んで表示される

### 個別 Pull / Push

1. 差分一覧から対象アイテムを選択
2. 右ペイン上部のボタンを使用:
    - **← Pull（サーバー → ローカル）**: サーバーのデータをローカルファイルに書き出す
    - **Push（ローカル → サーバー）→**: ローカルファイルのデータをサーバーに登録・更新する

### 全件 Pull / Push

- **全件 Pull ↓**: サーバーの全レコードをローカルに同期
- **全件 Push ↑**: ローカルの全ファイルをサーバーに同期

操作完了後、自動的に差分比較が再実行されます。

## 設定方法

CLI ツールと同じ設定ファイルおよび環境変数が使用できます:

| 方法                | 説明                                 |
| ------------------- | ------------------------------------ |
| `appsettings.json`  | アプリケーション配下のデフォルト設定 |
| `local.config.json` | ローカル設定（.gitignore 対象）      |
| 環境変数            | `EXTENSIONS_SYNC_` プレフィックス    |
| GUI 入力欄          | アプリ上部から直接入力（最優先）     |

## 差分ステータスの意味

| ステータス   | 色  | 説明                                 |
| ------------ | --- | ------------------------------------ |
| 一致         | 緑  | サーバーとローカルのコンテンツが同一 |
| 変更あり     | 橙  | サーバーとローカルに差分がある       |
| サーバーのみ | 青  | サーバーに存在するがローカルにない   |
| ローカルのみ | 紫  | ローカルに存在するがサーバーにない   |
